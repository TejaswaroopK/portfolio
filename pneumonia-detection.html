Pneumonia Detection and Classification

December 2023 - April 2024

Deep Learning Computer Vision TensorFlow Healthcare

Project Overview

Developed a Convolutional Neural Network (CNN) model to detect and classify pneumonia from chest X-ray images with 98% accuracy. The system helps radiologists identify pneumonia cases more efficiently, potentially saving lives through earlier detection.

Technical Details

Problem Statement

Pneumonia causes millions of deaths annually worldwide. Early detection is crucial but challenging due to the need for expert radiologists. This project aims to automate detection to improve diagnosis speed and accuracy.

Dataset

Used the Chest X-Ray Images (Pneumonia) dataset from Kaggle containing 5,863 validated images categorized into Normal, Bacterial Pneumonia, and Viral Pneumonia.

Model Architecture

Custom CNN with 5 convolutional layers, batch normalization, max pooling, and dropout layers to prevent overfitting. Final dense layers with softmax activation for classification.

Technologies Used

  • Python
  • TensorFlow/Keras
  • OpenCV for image preprocessing
  • Matplotlib/Seaborn for visualization
  • Flask for deployment prototype

Results & Impact

CNN model architecture diagram

Performance Metrics

  • Accuracy: 98%
  • Precision: 97%
  • Recall: 98%
  • F1-Score: 97.5%

Key Achievements

  • Reduced false negatives by 85% compared to baseline models
  • Implemented effective data augmentation to handle class imbalance
  • Developed Grad-CAM visualization to explain model decisions

Challenges & Solutions

Class Imbalance

The dataset had significantly more pneumonia cases than normal cases, which could bias the model.

Solution: Implemented weighted class loss and strategic data augmentation including rotations, flips, and brightness adjustments.

Overfitting

Initial models showed excellent training accuracy but poor validation performance.

Solution: Added dropout layers, batch normalization, and implemented early stopping with model checkpointing.
