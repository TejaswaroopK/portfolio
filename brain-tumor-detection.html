Brain Tumor Detection using CNN

June 2023 - November 2023

Deep Learning Medical Imaging TensorFlow Computer Vision

Project Overview

Developed a Convolutional Neural Network (CNN) model to detect brain tumors from MRI scans with 91% accuracy. The system classifies MRI images into four categories: no tumor, pituitary tumor, meningioma tumor, and glioma tumor.

Technical Implementation

Dataset

Used the Brain Tumor MRI Dataset from Kaggle containing 3,000+ T1-weighted contrast-enhanced MRI scans across four categories. Implemented extensive data augmentation to handle class imbalance.

Model Architecture

Custom CNN with 4 convolutional blocks (Conv2D + BatchNorm + ReLU + MaxPooling) followed by 3 dense layers. Used dropout (0.5) and L2 regularization to prevent overfitting.

Technologies

  • Python 3.8
  • TensorFlow 2.6
  • OpenCV for preprocessing
  • Matplotlib/Seaborn
  • Flask for demo deployment

Preprocessing

Applied skull stripping using BET, intensity normalization (N4 bias correction), and resizing to 224×224 pixels. Augmentations included rotation (±15°), zoom (0.9-1.1), and horizontal flips.

Results & Performance

Custom CNN architecture with layer dimensions

Evaluation Metrics

  • Overall Accuracy: 91.2%
  • Precision: 89-93% across classes
  • Recall: 88-92% across classes
  • F1-Score: 90.1% weighted average

Key Features

  • Grad-CAM visualization for model interpretability
  • DICOM image support for clinical compatibility
  • Optimized for low false negatives (sensitivity: 92%)

Challenges & Solutions

Class Imbalance

Glioma tumors were underrepresented (15% of dataset) compared to other classes.

Solution: Implemented class-weighted loss function and targeted oversampling of minority classes during augmentation.

Artifact Noise

Some scans contained motion artifacts and scanner noise.

Solution: Developed custom preprocessing pipeline with N4 bias correction and non-local means denoising.
